find_breadth_first/3

find_breadth_first(InitialVertex::vertex(), Predicate::predicate(), Feeder::feeder()) -> no_path_found | [vertex()]

Tries to find, breadth-first, the first vertex to satisfy specified predicate, from specified initial vertex, relying on the specified feeder function to obtain outbound edges of a vertex of interest.

Returns the path found, i.e. the ordered list of vertices, from the initial vertex to the found one.

find_breadth_first/4

find_breadth_first(InitialVertex::vertex(), Predicate::predicate(), Feeder::feeder(), UserData::basic_utils:user_data()) -> no_path_found | [vertex()]

Tries to find, breadth-first, the first vertex to satisfy specified predicate, from specified initial vertex, relying on the specified feeder function to obtain outbound edges of a vertex of interest.

Returns the path found, i.e. the ordered list of vertices, from the initial vertex to the found one.

The user data might be anything, from 'undefined' to a description of the graph structure (see graph_utils_test.erl).
