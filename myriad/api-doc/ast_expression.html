Description

Module in charge of handling expressions defined with an AST.

See http://erlang.org/doc/apps/erts/absform.html for more information.

Data Types

ast_expression()

ast_expression() = ast_base:ast_element()

The description of an expression in an AST, with in-file location information.

For example '{integer, {97,1}, 2}' or '{match, 117, {var,117,'A'}, {atom, 117, foobar}}', etc.

Note that an expression is different from a pattern: even if they share at least some types of forms, they are to be interpreted differently (e.g. their sub-elements are of the same kind as they are, and at least some rules differ).
