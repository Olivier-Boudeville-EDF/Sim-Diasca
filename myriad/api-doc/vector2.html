Module implementing the support for 2D vectors.

Description

Module implementing the support for 2D vectors.

See also: - the corresponding points (in point2.erl) and matrices (in matrix2.erl) - the (unspecialised) vectors of arbitrary dimensions, in vector.erl - the various 2D extra services in linear_2D.erl
