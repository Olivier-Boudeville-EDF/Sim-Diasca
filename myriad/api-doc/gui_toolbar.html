Description

Gathering of various facilities for toolbar management.

A tool bar is a bar of buttons and/or other controls usually placed below the menu bar in a frame.

Use our gui_frame_bars_test.erl test in order to display all known tools.

toolbar()

abstract datatype: toolbar()

A bar of image-based buttons and/or other controls usually placed below the menu bar of a frame.

A toolbar emits menu commands in the same way that a frame menubar does.
