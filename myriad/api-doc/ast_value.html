ast_atomic_literal()

ast_atomic_literal() = {atom, file_loc(), atom()} | {char, file_loc(), char()} | {float, file_loc(), float()} | {integer, file_loc(), integer()} | {string, file_loc(), ustring()}

The description of an immediate value (atomic literal) in an AST, with in-file location information (line/column).

Ex: nil, in {nil,{33,2}} for [] at column 2 of line #33.
