Module implementing the support for points of arbitrary dimension.

Description

Module implementing the support for points of arbitrary dimension.

See also: - the corresponding arbitrary-dimensioned vectors (in vector.erl) and matrices (in matrix.erl) - the specialised points, such as points{2,3,4}.erl
