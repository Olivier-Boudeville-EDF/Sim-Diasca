plane3()

plane3() = {normal3(), factor()}

A plane, whose general equation is: A.x + B.y + C.z + D = 0, where:

- P={x,y,z} is a point belonging to this plane - N=[A,B,C] is a (non-necessarily unit) normal vector to this plane - D= -A.x0 - B.y0 - C.z0

See http://mathworld.wolfram.com/Plane.html.

So a plane may be described as (N,D).
