Module implementing the support for 4D points.

Description

Module implementing the support for 4D points.

See also: - the corresponding vectors (in vector4.erl) and matrices (in matrix4.erl) - the (unspecialised) points of arbitrary dimensions, in point.erl - the various 4D services in linear_4D.erl
