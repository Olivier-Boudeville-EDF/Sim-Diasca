Function Details

create_presized_static_display/3

create_presized_static_display(Label::label(), Font::font(), Parent::parent()) -> {static_text_display(), precise_text_extent()}

Creates a static text display, based on the specified label and on a precomputed size, determined thanks to the specified font, which is associated to it. Returns the created static display, together with its precise text extent.

This function may be useful as, in some cases, even the rendering if a single-line label in a panel may be wrong, being cropped for some unknown reason (presumably a wxWidgets bug).

create_presized_static_display/4

create_presized_static_display(Label::label(), Options::maybe_list(static_display_option()), Font::font(), Parent::parent()) -> {static_text_display(), precise_text_extent()}

Creates a static text display, based on the specified label, (non-size) option(s) and its precise text extent, determined thanks to the specified font, which is associated to it.

Note that the display height may be higher than the one of the actual text, due to the margin taken for letters possibly going below the baseline (like 'g').

This function may be useful as, in some cases, even the rendering if a single-line label in a panel may be wrong, being cropped or extended for some unknown reason (presumably a wxWidgets bug).
