pep8_class_to_pep8_module/1

pep8_class_to_pep8_module(Classname::pep8_classname() | ustring()) -> pep8_class_module()

Deduces the name of the Python module from the name of the Python class (possibly prefixed with package names) that it implements, according notably to the naming conventions adopted in PEP 8.

Ex: 'Partner__TransportModel__MyFoobarExample' resulting in 'partner.transport_model.my_foobar_example'.
