Description

Support for the Protocol Buffer facilities, a.k.a. Protobuf.

We generally prefer using the proto3 version (rather than the proto2 one).

See protobuf_support_test.erl for the corresponding test, and http://myriad.esperide.org/#about-protobuf for further information.

message_type()

message_type() = atom()

The name of a type of Protobuf messages, as defined in a specification file.

For example, if defining a "Person" message type in a "myriad.protobuf.test" package, the resulting record is #myriad_protobuf_test_person{}, corresponding to a myriad_protobuf_test_person() type.

See myriad_example.proto for a test example.
