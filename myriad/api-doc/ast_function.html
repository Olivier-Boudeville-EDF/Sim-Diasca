get_located_forms_for(FunctionExportTable::function_export_table(), FunctionTable::function_table()) -> {[located_form()], [located_form()]}

Returns a pair made of (two) lists of located forms regarding exports.

These lists of located forms correspond to:

- all the function export declarations (possibly automatically enriched) that are described in the specified function export table

- all the function definitions and specs that are described in the specified function table
