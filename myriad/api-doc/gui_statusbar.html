field_width()

field_width() = integer()

The width of a field in a status bar.

If positive, designates a fixed width, in pixels. If negative, designates a width ratio among all variable-width fields.

status_bar()

abstract datatype: status_bar()

A thin space at the bottom of a frame where texts are typically displayed.

They are very useful to trace events, and can be split in separate fields.

status_bar_style() = normal | flat | raised | sunken
