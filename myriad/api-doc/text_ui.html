This is the most basic, line-based monochrome textual interface, directly in raw text, with no cursor control.

Description

This is the most basic, line-based monochrome textual interface, directly in raw text, with no cursor control.

See: - text_ui_test.erl for the corresponding test - gui.erl for a graphical counterpart

See also: trace_utils.erl for another kind of output.
