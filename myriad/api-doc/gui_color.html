rgb_hexastring()

rgb_hexastring() = ustring()

A RGB color that is encoded based on 6 hexadecimal digits in a string, with a "#" prefix (hence is not a text_utils:hexastring/0).

For example "#3ab001" corresponds to lightgreen.

Possibly used for HTML content, by gnuplot, etc.
