Description

Module in charge of handling patterns defined with an AST.

See the "7.3 Patterns" section of http://erlang.org/doc/apps/erts/absform.html for more information.

Data Types

ast_pattern()

ast_pattern() = ast_base:ast_element()

The description of a pattern in an AST, with in-file location (line/column) information.

For example: {match,12,X,Y}.

Too many patterns to fully specify all (see "7.3 Patterns").

Note: a pattern is different from an expression: even if they share at least some types of forms, they are to be interpreted differently (e.g. their sub-elements are of the same kind as they are, and at least some rules differ).

ast_pattern_sequence()

ast_pattern_sequence() = [ast_pattern()]

The description of a sequence of patterns in an AST, with in-file location information.

A pattern sequence is simply an (ordered) list of patterns.

For example: [{match,12,X,Y}].
