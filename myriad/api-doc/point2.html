Module implementing the support for 2D points.

Description

Module implementing the support for 2D points.

See also: - the corresponding vectors (in vector2.erl) and matrices (in matrix3.erl) - the (unspecialised) points of arbitrary dimensions, in point.erl - the various 2D services in linear_2D.erl
