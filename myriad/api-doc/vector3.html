Module implementing the support for 3D vectors.

Description

Module implementing the support for 3D vectors.

See also: - the corresponding points (in point3.erl) and matrices (in matrix3.erl) - the (unspecialised) vectors of arbitrary dimensions, in vector.erl - the various 3D extra services in linear_3D.erl
