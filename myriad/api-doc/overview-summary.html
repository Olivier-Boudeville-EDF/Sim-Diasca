Overview

Ceylan-Myriad is an Erlang library that gathers the general-purpose base constructs that we found useful for most developments.

This website presents the documentation of the API (Application Programming Interface) of Myriad version 1.0.22 (as of Tuesday, January 11, 2022), in order to ease the developments relying on the Myriad toolbox.

See also the general description of the services offered by Myriad.
