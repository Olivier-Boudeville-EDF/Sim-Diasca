Gathering of various facilities regarding the management of SQL databases.

Description

Gathering of various facilities regarding the management of SQL databases.

The current implementation is mostly geared towards the use of:

- SQLite3 (an optional dependency), through the erlang-sqlite3 binding

- PostgreSQL, through either its command-line client (psql; the least recommended approach) or the epgsql https://github.com/epgsql/epgsql binding

See sql_support_test.erl for the corresponding test.
