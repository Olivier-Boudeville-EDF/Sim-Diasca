Description

Module in charge of handling clauses defined within an AST.

Refer to the "7.5 Clauses" section of http://erlang.org/doc/apps/erts/absform.html for more information.

ast_clause()

ast_clause() = ast_function_clause() | ast_if_clause() | ast_case_clause() | ast_try_clause() | ast_catch_clause()

There are 5 different kinds of clauses in an AST: - function clauses - if clauses - case clauses - try clauses - catch clauses

One may note they actually all obey the same structure (same quintuplet).

ast_function_clause()

ast_function_clause() = ast_generic_clause()

Describes a function clause in an AST:

"If C is a function clause ( Ps ) -> B, where Ps is a pattern sequence and B is a body, then Rep(C) = {clause, FILE_LOC, Rep(Ps), [], Rep(B)}.

If C is a function clause ( Ps ) when Gs -> B, where Ps is a pattern sequence, Gs is a guard sequence and B is a body, then Rep(C) = {clause, FILE_LOC, Rep(Ps), Rep(Gs), Rep(B)}."

transform_catch_clause/2

transform_catch_clause(Clause::ast_catch_clause(), Transforms::ast_transforms()) -> {ast_catch_clause(), ast_transforms()}

Transforms specified catch clause.

Catch clause with no variable, with or without a guard sequence (1/4 and 3/4):

- "If C is a catch clause P -> B, where P is a pattern and B is a body, then Rep(C) = {clause, FILE_LOC, [Rep({throw,P,_})], [], Rep(B)}."

- "If C is a catch clause P when Gs -> B, where P is a pattern, Gs is a guard sequence, and B is a body, then Rep(C) = {clause, FILE_LOC, [Rep({throw,P,_})], Rep(Gs), Rep(B)}."

transform_function_clause/2

transform_function_clause(Clause::ast_function_clause(), Transforms::ast_transforms()) -> {ast_function_clause(), ast_transforms()}

Transforms specified function clause.

Handled the same, with or without guard(s), as a guard sequence may be empty:

- without: "If C is a function clause ( Ps ) -> B, where Ps is a pattern sequence and B is a body, then Rep(C) = {clause, FILE_LOC, Rep(Ps), [], Rep(B)}."

- with: "If C is a function clause ( Ps ) when Gs -> B, where Ps is a pattern sequence, Gs is a guard sequence and B is a body, then Rep(C) = {clause, FILE_LOC, Rep(Ps), Rep(Gs), Rep(B)}."
