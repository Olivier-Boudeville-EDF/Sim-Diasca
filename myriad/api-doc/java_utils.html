Function Details

classname_to_bytecode_filename/1

classname_to_bytecode_filename(Classname::java_classname() | ustring()) -> java_bytecode_filename()

Deduces the (root) name of a Java bytecode file from the name of the class it implements, according to the naming conventions used by the language.

With Java, both names are identical except the extension, hence we just check if the name looks CamelCased, i.e. if at least its first letter is in upper case.
