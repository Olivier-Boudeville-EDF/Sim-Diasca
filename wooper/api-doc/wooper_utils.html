get_java_package_and_class_for/1

get_java_package_and_class_for(WOOPERClassname::wooper:classname()) -> java_utils:java_fully_qualified_classname()

Returns (as atoms) the Java package (if any) and class that correspond to the specified WOOPER classname.

So for example a WOOPER classname equal to 'class_BigPackage__MyPackage__MyExample' is to be translated into: { 'bigpackage.mypackage', 'MyExample' }, while for 'class_MyExample' we have { undefined, MyExample } returned.

Note: no Java package shall be named 'undefined'.
